# fract-ol

# How to run program?

Commands:
    
    ./fractol Mandelbrot
    ./fractol Julia
    ./fractol Burning_Ship
    ./fractol Celtic_mandelbar
    ./fractol Perpendicular_buffalo
    
Control:

    Zoom: scroll;
    
    Moving: arrows;
    
    Switch color scheme: c;
    
    Basic set: r;
    
    For Julia set:
    
    Change a parameter:     press and hold the left
                        mouse button and move mouse
